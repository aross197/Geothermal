# Springhill Geothermal Single-Loop System

**A Residential Geothermal Innovation for Springhill, Nova Scotia**

An innovative "single-loop" direct-DC mini-split geothermal system designed to harness the stable \~18°C flooded mine water in Springhill for efficient, year-round heating and cooling.

### Project Overview

This system is built on the **original single-loop concept** invented by **Ralph Ross** (Ross Refrigeration). It has been further developed through collaboration with:

- **Ralph Ross** – Original Concept, Schematic & Core Invention
- **Rob Charest** – Product Development & Collaborative Invention
- **Andrew Ross** – Enhanced Closed-Loop Design, Solar Integration & Project Lead

The design overcomes the limitations of air-source heat pumps in cold climates by using the abundant thermal resource from Springhill’s historic coal mines. It features:
- Direct-DC power operation
- Micro-hydro self-generation from return water
- Optional solar PV integration
- Zoned comfort using four 4-way ceiling cassettes
- Closed-loop glycol option for improved durability and permitting

### Key Features

- Stable mine water temperature (\~18–20°C, potentially warmer at depth)
- High-efficiency R-32 refrigerant loop with plate heat exchanger (HX1)
- Variable-speed ceiling cassettes with \~400 CFM/ton airflow recommendation
- Closed-loop piggyback configuration (Andrew Ross enhancement)
- Micro-hydro + solar for greater energy independence

### How the System Works

1. **Open-Loop Baseline** (Ralph Ross original): Mine water transfers heat to the refrigerant loop via a plate heat exchanger. Chilled return water drives a micro-hydro turbine for on-site DC power.
2. **Closed-Loop Enhancement**: A sealed glycol loop exchanges heat with the mine water without direct contact.
3. **Solar Integration**: Direct-DC solar PV supplements micro-hydro generation.
